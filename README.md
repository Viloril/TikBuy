📱 О приложении
TikBuy - это современное мобильное приложение для создания и управления списками покупок. Приложение помогает организовать процесс покупок, делая его удобным и эффективным.

✨ Основные возможности
✅ Создание списков покупок - создавайте неограниченное количество списков для разных целей

📝 Управление товарами - добавляйте, редактируйте и удаляйте товары в каждом списке

📊 Отслеживание прогресса - визуальный индикатор выполнения покупок

🏷️ Категоризация - распределяйте товары по категориям (продукты, бытовая химия и т.д.)

✅ Отметка покупок - отмечайте купленные товары одним нажатием

🗑️ Автоматическое завершение - интеллектуальное завершение списка при покупке всех товаров

💾 Локальное хранение - все данные сохраняются в локальной базе данных SQLite

🚀 Технологии
Язык программирования: Java

Платформа: Android

База данных: SQLite

UI/UX: Material Design

Архитектура: MVC
